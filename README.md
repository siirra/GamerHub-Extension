# GamerHub: Ultimate Steam & CS2 Dashboard

GamerHub is an open-source Chrome Extension designed for Steam power users and CS2 traders.

## ✨ Features
- **Inventory Manager**: View CS2 item prices (via Skinport) and Float values.
- **Social Tracker**: Track friends' status and get notifications when they start playing.
- **Faceit Integration**: View Faceit levels and ELO directly.
- **Deal Scanner**: Track game deals and price drops via CheapShark.

## 🛠️ Setup
 **Extension**: Load the `/GamerHub-Extension` folder as an unpacked extension in Chrome.
